# Outreach targets: Media & PR

Companion to [Backlinks & promotion playbook](../backlinks-and-promotion.md) §2.3.

**Disclaimer:** Journalists owe you nothing. Lead with **news value**, **human stories**, or **data**—not “please link my site.” Use these services and outlets as **starting points**; verify beats and contacts on each site.

**Concrete prospect list (outlets + ESN / Meetup):** [07-concrete-outreach-prospects.md](./07-concrete-outreach-prospects.md) §G, **§M** (tech / startup media).

## A. Journalist request platforms (HARO-style)

| Service | URL | What it is | How to automate / scale | Tips |
|---------|-----|------------|---------------------------|------|
| **HARO (Help a Reporter Out)** | [helpareporter.com](https://www.helpareporter.com/) | Email digest of journalist queries | Filter inbox rules: keywords `Netherlands`, `expat`, `visa`, `EU`, `relocation`, `remote work` | Reply **fast** (same day), **short**, credentials upfront |
| **Qwoted** | [qwoted.com](https://www.qwoted.com/) | Similar to HARO; tech/finance skew | Email alerts + dashboard; API on paid tiers | Complete profile with **NL expertise** |
| **SourceBottle** | [sourcebottle.com](https://www.sourcebottle.com/) | AU/UK skew; some EU | Newsletter filter | Useful for **UK→NL** mobility stories |
| **ResponseSource** (UK) | [responsesource.com](https://www.responsesource.com/) | UK media requests | Paid; filter by keywords | Good for **Brexit / UK nationals in NL** angles |
| **Twitter/X #journorequest** | Search: `#journorequest Netherlands` or `Netherlands journalist` | Ad hoc | Set **saved search** + notifications (careful with noise) | Very high noise; still occasional gems |

**Semi-automation:** Forward HARO to a dedicated label; nightly script (Apps Script / Zapier) that lists subject lines containing your keywords into Slack for human triage.

| Name / type | Resource / link page | Why it fits | Who to contact | Example pitch hook |
|-------------|----------------------|-------------|----------------|---------------------|

| **Search Substack:** `Netherlands` + `expat` | Individual newsletter **About** | Niche audiences, high trust | Author email on Substack | “Sponsor a section or contribute a guest post with one canonical link.” |
| **Relocation / HR podcasts** | Search `global mobility podcast` | Employer audience | Producer email on show site | “Topic: orientation year changes—non-salesy.” |

## Outreach templates

Journalists want **speed**, **credentials**, and **usable quotes**—not “please backlink.” The backlink often appears if you’re cited; don’t ask for the link in the first sentence.

**Placeholders:** `OUTLET_NAME`, `JOURNALIST_NAME`, `HARO_QUERY_SUMMARY`, `YOUR_CREDENTIAL`, `QUOTE_READY_SENTENCE`, `YOUR_PAGE_URL`, `NEWS_HOOK`, `PODCAST_NAME`, `EPISODE_OR_TOPIC`.

### T1 — HARO / Qwoted / SourceBottle reply (paste into email body or platform form)

**Subject line:** Often fixed by the platform — use their **Reply** subject if provided.

```
Query: HARO_QUERY_SUMMARY

YOUR_CREDENTIAL

Answer:
QUOTE_READY_SENTENCE

Context we use with readers: YOUR_PAGE_URL (fact-checked; links to IND/official sources).

Available for a quick call today if useful — [timezone]. WhatsApp / Signal: [optional]

[Your name]
[Title / org]
[Email]
[City, NL if relevant]
```

**Rules:** Respond **same day**; **≤200 words** unless they ask for more; no attachments unless requested.

### T2 — Cold pitch to editor (news / explainer after a policy story)

**Subject:** Explainer offer re: NEWS_HOOK — YOUR_SITE_NAME

```
Hi JOURNALIST_NAME / editors,

Following OUTLET_NAME’s coverage of NEWS_HOOK, readers may want a step-by-step timeline (what changes when, who it affects).

We published YOUR_PAGE_URL — we can provide:
• 2–3 verified bullet points for your next piece
• Or a short guest explainer if you run those (your guidelines apply)

YOUR_CREDENTIAL

Best,
[Your name]
[Email]
```

### T3 — Tips / newsdesk (anonymous desk)

**Subject:** Tip: NEWS_HOOK — background + source

```
Hello OUTLET_NAME newsdesk,

Background for NEWS_HOOK: [2 sentences factual summary].

We’ve summarised the timeline for non-specialists here: YOUR_PAGE_URL

Happy to be attributed as “[Your name], YOUR_SITE_NAME” or to point you to primary sources on IND.nl.

[Your name]
[Phone]
```

### T4 — Podcast host (guest pitch)

**Subject:** Guest idea — TOPIC (Netherlands / expats)

```
Hi [Host name],

I listened to PODCAST_NAME on EPISODE_OR_TOPIC — [one genuine sentence about what landed].

I’d love to join for 15–20 minutes on TOPIC: [one-line angle]. I can share mistakes new arrivals make that employers and schools see every year — no sales pitch.

Background: YOUR_CREDENTIAL
Reference material we’d promote with the episode: YOUR_PAGE_URL

Open to pre-interview questions by email.

Best,
[Your name]
[Email]
```

### T5 — Newsletter author (Substack / industry)

**Subject:** Guest post or quote — TOPIC

```
Hi [Author name],

Your piece on [their recent topic] resonated — especially [specific point].

I write YOUR_SITE_NAME and could offer either:
(1) A short guest section on TOPIC with one outbound link to YOUR_PAGE_URL, or
(2) A pull quote for your issue with attribution only

Totally fine if you only do paid placements — if so, who should I speak to?

Thanks,
[Your name]
[Email]
```

### T6 — Follow-up after they used your quote (relationship, not demand)

**Subject:** Thanks — and a permalink if helpful

```
Hi JOURNALIST_NAME,

Thanks for including the context in your piece on NEWS_HOOK.

If readers ask for the full checklist, the permalink is YOUR_PAGE_URL — happy to update it if rules change.

[Your name]
```
