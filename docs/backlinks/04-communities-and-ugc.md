# Outreach targets: Communities & UGC

Companion to [Backlinks & promotion playbook](../backlinks-and-promotion.md) §2.4.

**Disclaimer:** Most communities **ban** or **limit** self-promotion. **Participation first.** Links are often **nofollow** or **temporary**—value here is **trust**, **traffic**, and **ideas** for content, not raw SEO manipulation. Read each community’s rules before posting.

**Concrete entry points (ESN sections, Meetup search, InterNations):** [07-concrete-outreach-prospects.md](./07-concrete-outreach-prospects.md) §G (rows 74–76).

## A. Reddit (English, Netherlands-focused)

| Subreddit | URL | Rules to respect | How to add value | Link policy reality |
|-----------|-----|------------------|------------------|---------------------|
| **r/Netherlands** | [reddit.com/r/Netherlands](https://www.reddit.com/r/Netherlands/) | Read sidebar + pinned **rules**; no spam | Answer **specific** questions; offer summary in comment, link **only** if OP asks or wiki allows | Mods remove obvious promo; **karma** and history matter |
| **r/Amsterdam** | [reddit.com/r/Amsterdam](https://www.reddit.com/r/Amsterdam/) | Stricter on housing ads | Same—practical answers | Same |
| **r/thenetherlands** | [reddit.com/r/thenetherlands](https://www.reddit.com/r/thenetherlands/) | Related; check overlap | Same | Same |
| **r/ExpatFIRE** / **r/expats** | Search on Reddit | Global; NL threads appear | Answer when **NL-specific** | Often **nofollow**; still useful for feedback |
| **Country subs** (e.g. Americans in NL) | Search `[country] Netherlands reddit` | Varies | Extremely sensitive to ads | **Never** cold-post your site |

**Automation (allowed):** Keyword alerts (F5bot, Google Alerts on `site:reddit.com Netherlands visa`) → **human** decides whether to reply. **Do not** bot-post.

## B. Facebook groups

Facebook links are typically **nofollow** and groups are **private**—track **referrals** in GA4 under **social / referral** if shareable.

| Group type | How to find | Approach | Risk |
|------------|-------------|----------|------|
| **“Americans in the Netherlands”** (and similar) | Facebook search | Join, **lurk 2 weeks**, answer questions **without** link; add link only when rules allow “resource Saturday” etc. | Ban for spam |
| **City parent / housing groups** | Same | Housing threads—**extreme** sensitivity to spam | High ban risk |
| **University alumni groups** | Search `[University] alumni Netherlands` | Often allow **resource** threads with mod approval | Ask mod first |

**Table of example search strings (not group URLs—groups change):**

| Search query | Intent |
|--------------|--------|
| `Americans in Amsterdam Facebook group` | US expats |
| `British expats Netherlands Facebook` | Post-Brexit themes |
| `International parents Amsterdam` | Schools + gemeente |
| `Highly skilled migrant Netherlands` | IND / work permit |

## Summary — community DO / DON’T

| DO | DON’T |
|----|--------|
| Build karma / reputation first | Drop link in first message |
| Ask mods about **resource** threads | Use multiple accounts |
| Screenshot **high-upvote** answers to improve your own guide | Buy upvotes or engagement |
| Track which questions repeat → **new content** | Copy-paste same reply everywhere |

## Outreach templates

Communities are **not** email-first. Use **comments**, **mod mail**, or **organiser DMs**. Never lead with your URL.

**Placeholders:** `SUBREDDIT_OR_GROUP`, `USER_QUESTION_SUMMARY`, `YOUR_PAGE_URL`, `YOUR_SITE_NAME`, `DISCLOSURE_LINE` (e.g. “I work on YOUR_SITE_NAME, a free guide site.”).

### T1 — Reddit: helpful answer **without** link (default)

```
[Opening empathy one line — e.g. “Municipality first is the usual order, but it depends on whether you already have a BRP address.”]

Steps that usually apply:
1. [Bullet]
2. [Bullet]
3. [Bullet]

Official source: [link to IND or gemeente — not your site]

If anything contradicts what your gemeente told you, trust them.

DISCLOSURE_LINE [only if you’re officially affiliated and sub rules require it]
```

### T2 — Reddit: answer **with** link (only if OP asks for a guide, or wiki contribution)

```
[Same structure as T1 — value in the comment]

We also keep a plain-language checklist here: YOUR_PAGE_URL — it links back to IND and [official source]. Hope it helps; happy to correct anything that’s outdated.

DISCLOSURE_LINE
```

### T3 — Reddit: mod mail (asking to add to wiki / sidebar)

**To:** Subreddit mod mail

```
Hi mods of r/[name],

I’m a regular contributor [optional: link to 2–3 past helpful comments if true].

Would you consider adding YOUR_PAGE_URL to the wiki under [suggested section] for [topic]? It’s non-commercial, links to official sources, and we update when rules change.

If you prefer not to link external sites, no problem — thanks for maintaining the sub.

[Username]
```

### T4 — Facebook group: ask a moderator first

**Messenger / post to admins (if allowed)**

```
Hi admins,

I run YOUR_SITE_NAME (free guides for [city/NL], no paid relocation service). Would it be OK to share YOUR_PAGE_URL in a comment when someone asks about TOPIC, or on a resources thread if you run one?

Happy to follow whatever promo rules you have.

Thanks,
[Your name]
```

### T5 — Facebook: reply in thread (after establishing presence)

```
[Direct answer to their question in 3–5 short paragraphs or bullets]

If useful, we wrote this up with links to [official source]: YOUR_PAGE_URL — only sharing because you asked about USER_QUESTION_SUMMARY.

DISCLOSURE_LINE
```

### T6 — Discord / Slack (public server, when someone asks)

```
[Short answer in-channel]

More detail + official links collected here: YOUR_PAGE_URL

DISCLOSURE_LINE
```

### T7 — Meetup / Eventbrite organiser (speaker offer)

**Subject:** Offer: short talk — TOPIC for [Meetup name]

```
Hi [Organiser name],

I’m [name] from YOUR_SITE_NAME. I’d be glad to do a 15-minute Q&A on TOPIC for your [Amsterdam expat / international] meetup — no charge, no pitch beyond introducing the site in one line.

If that fits, we can add YOUR_PAGE_URL to the event description as optional reading.

Best,
[Your name]
[Email]
```

### T8 — Quora answer (disclosure required in profile or answer per Quora policy)

**Opening**

```
[Direct answer first — 2–3 paragraphs]

For a step-by-step with citations to IND and the gemeente: YOUR_PAGE_URL

DISCLOSURE_LINE (e.g. “I contribute to YOUR_SITE_NAME.”)
```

### T9 — InterNations / forum (respect ToS; often stricter)

```
[Neutral answer]

Further reading (third-party guide with official links): YOUR_PAGE_URL

DISCLOSURE_LINE
```
