# Outreach targets: Directories & roundups

Companion to [Backlinks & promotion playbook](../backlinks-and-promotion.md) §2.6.

**Disclaimer:** Many directories exist **only for SEO** and are **low or negative value**. Prioritise **human-curated** lists, **local** guides, and **editorial** roundups. **Verify** whether links are **nofollow**—they can still drive **traffic**.

## B. “Best resources” roundups (pitch the author)

Use search operators to find **pages** that already list resources—then ask for **inclusion** with a **specific** reason.

| Search operator | What you find |
|-----------------|---------------|
| `"moving to the Netherlands" resources` | Blog roundups |
| `"expat" "Netherlands" "useful links"` | Link lists |
| `site:.edu "Netherlands" "study abroad" links` | University societies (rare) |
| `site:.nl english guide internationals` | Dutch org English hubs |

**Example outreach table (fill in URLs you discover):**

| Step | Action |
|------|--------|
| 1 | Run 5–10 operators; export top 20 URLs to a sheet |
| 2 | Column: **Page title**, **Last updated** (rough), **Email** (hunter.io / site contact) |
| 3 | Column: **Gap** (“No section on health insurance timing”) |
| 4 | One email per row; track response |

## Outreach templates

Directories mix **editorial**, **paid listings**, and **guest posts**—ask explicitly which applies. For **roundup** authors, prove you **fit their category** and **their audience**.

**Placeholders:** `SITE_NAME`, `THEIR_ROUNDUP_URL`, `YOUR_PAGE_URL`, `YOUR_SITE_NAME`, `LIST_CATEGORY`, `AUTHOR_NAME`, `TOPIC`.

### T1 — Expat directory / guide site (listing or editorial)

**Subject:** Listing / content enquiry — YOUR_SITE_NAME (Netherlands / expats)

```
Hi [Editorial or partnerships team],

We run YOUR_SITE_NAME — free guides and tools for people moving to and living in the Netherlands (registration, BSN, insurance basics, visas at a non-legal level). Primary URL: YOUR_PAGE_URL

Could you tell me:
1. Whether you accept new listings in [category], and
2. Whether that’s editorial, sponsored, or mixed?

If you have media kit / rate card, please send it. We’re happy to meet editorial guidelines.

Best,
[Your name]
[Email]
```

### T2 — “Best resources” roundup author (specific + humble)

**Subject:** Suggestion for your list on THEIR_ROUNDUP_URL

```
Hi AUTHOR_NAME,

Your roundup THEIR_ROUNDUP_URL was useful — especially the section on [specific item].

We maintain YOUR_PAGE_URL on LIST_CATEGORY; it’s updated when IND rules shift and only links to official sources. If you’re still updating the post, would you consider adding it?

Totally understand if you keep the list minimal.

Thanks,
[Your name]
[YOUR_SITE_NAME]
```

### T3 — Guest post pitch (blog with guidelines)

**Subject:** Guest post pitch — TOPIC for SITE_NAME readers

```
Hi [Editor],

Pitch: [One-sentence headline].

Angle: [2 bullets max — what the reader learns]

We’d link to one canonical resource: YOUR_PAGE_URL
Bio: [one line — not a law firm / not financial advice if true]

Please send contributor guidelines if this fits your editorial calendar.

Best,
[Your name]
[Email]
```

### T4 — Product / tool directory (AlternativeTo, Product Hunt style)

**Short description (often character-limited)**

```
YOUR_SITE_NAME — Free Netherlands relocation guides and tools: registration, BSN, mandatory insurance, visa timelines (non-legal). Links to IND and official sources.
```

**Longer “maker” comment (Product Hunt)**

```
We built this because [problem]. Best for [who]. Not a replacement for IND or legal advice — we send people to official pages for decisions.

Try: YOUR_PAGE_URL

Happy to answer questions.
```

### T5 — Invest in Holland / enterprise (only if you have B2B angle)

**Subject:** Resource for international talent — partnership enquiry

```
Dear [Programme team],

[Company / YOUR_SITE_NAME] helps internationals navigate first-step administration in the Netherlands. We’re exploring whether our materials could support [employers / talent programmes] you work with.

Overview: YOUR_PAGE_URL

Could you point us to the right partnership or communications contact?

Kind regards,
[Your name]
[Email]
```

### T6 — Follow-up (roundup / directory, one time)

**Subject:** Re: Suggestion for your list on THEIR_ROUNDUP_URL

```
Hi AUTHOR_NAME,

Quick bump on my note from [date]. If the list is closed, no worries — thanks for maintaining a helpful resource.

[Your name]
```
