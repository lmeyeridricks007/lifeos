# Shared outreach templates (optional)

Use this file only if you want to **draft variants** with placeholders. For **ready-to-send** copy (Lee / expatcopilot.com), use [lee-ready-to-send-part-1-of-3.md](./lee-ready-to-send-part-1-of-3.md) and the other parts in [README.md](./README.md).

Replace placeholders before sending:

| Placeholder | Meaning |
|-------------|---------|
| `{{SITE_URL}}` | Your site origin, **no** trailing slash |
| `{{BEST_PATH}}` | Path from roster, e.g. `/netherlands/visa/student-visa/` |
| `{{ORG}}` | Organisation name |
| `{{REACH_HINT}}` | From §07 “Reach via” column |
| `{{THEIR_PAGE_TOPIC}}` | Short note tied to their site (section you read) |

Full link in body: `{{SITE_URL}}{{BEST_PATH}}` (paths in the repo use trailing slashes).

**Subject lines** are suggestions; adapt to their tone.
